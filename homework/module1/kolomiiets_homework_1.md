## Завдання 2. Робота з документацією

Виконано команди:
```bash
man ls
help cd
man cat
man man
cp --help
mv --help
```

### 2.1. Який ключ `ls` показує приховані файли?

Ключ `-a` (`--all`) — показує всі записи, включно з файлами, що починаються з крапки (`.` та `..`, а також приховані `.bashrc`, `.config` тощо).

Близький варіант — `-A` (`--almost-all`): те саме, але без `.` і `..`.

### 2.2. Який ключ у `cat` нумерує рядки?

Ключ `-n` (`--number`) — нумерує всі рядки виводу.

Варіант `-b` (`--number-nonblank`) — нумерує лише непорожні рядки.

### 2.3. Чим відрізняються `man` і `--help`?

| `man <команда>` | `<команда> --help` |
|---|---|
| Окрема система сторінок документації (manual pages), що зберігаються у базі `man` і відкриваються через pager (`less`). | Вбудована в саму програму опція, яка виводить коротку довідку у `stdout`. |
| Повна, розгорнута документація: `NAME`, `SYNOPSIS`, `DESCRIPTION`, усі опції, `EXAMPLES`, `SEE ALSO`, `AUTHOR`. | Короткий список опцій і синтаксис — зазвичай одна-дві екранні сторінки. |
| Організована в розділи (1 — утиліти, 2 — системні виклики, 3 — бібліотечні функції, 5 — формати файлів тощо). | Формат і обсяг залежать виключно від розробника конкретної програми. |
| Може бути недоступна, якщо пакет `man-pages` не встановлено або сторінку не включено. | Працює лише якщо програма сама підтримує `--help` (у вбудованих команд bash цієї опції зазвичай немає — там `help <команда>`). |

## Завдання 3. Міні-сценарій

Сценарій із 8 команд із використанням тем 1–5 (`pwd`, `ls`, `cd`, перенаправлення `>`, `man`):

```bash
cd ~/Downloads          # 1. перейти в каталог
pwd                     # 2. перевірити поточний шлях
> scenario_file.txt     # 3. створити один файл
ls                      # 4. переглянути список файлів
cd /tmp                 # 5. перейти в інше місце
pwd                     # 6. перевірити, що справді в /tmp
cd -                    # 7. повернутись назад (у попередній каталог)
man ls                  # 8. переглянути документацію однієї з команд
```

Вивід (без `man ls`, який відкриває pager):
```
$ cd ~/Downloads
$ pwd
/home/leg-p21555/Downloads
$ > scenario_file.txt
$ ls -la scenario_file.txt
-rw-rw-r-- 1 leg-p21555 leg-p21555 0 апр 13 22:01 scenario_file.txt
$ cd /tmp
$ pwd
/tmp
$ cd -
/home/leg-p21555/Downloads
$ pwd
/home/leg-p21555/Downloads
```
